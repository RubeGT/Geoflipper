# Geoflipper
Geoflipper is a free geography guessing game inspired by GeoGuessr. Players are dropped into a random location using Google Maps Street View and must figure out where they are based on the environment around them.

Built with the Google Maps API and Google Maps JavaScript services, Geoflipper delivers an accessible browser-based experience without requiring a paid subscription.

Features
Random Street View locations
Interactive world map guessing
Distance-based scoring system
Fast and lightweight web experience
Powered by Google Maps APIs and JavaScript services
Tech Stack
JavaScript
Google Maps JavaScript API
Google Street View Services
HTML/CSS
